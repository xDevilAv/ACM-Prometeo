El **i-gel** es un tipo de **vía aérea supraglótica (SGA)** utilizado para asegurar la vía aérea en pacientes inconscientes.

# Función
El i-gel se utiliza para asegurar la vía aérea en casos de **{% include hl.html t="Airway Collapse" %} grave**, abriendo físicamente la vía aérea y permitiendo que el aire pase a través del tubo hacia la vía aérea, además de sellarla para evitar la aspiración de contenido gástrico.

Cuando está insertado, impedirá establecer la **{% include hl.html t="Recovery Position" %}**.

El i-gel evita que el vómito provoque **{% include hl.html t="Airway Obstructions" l="Airway_Obstruction" %}**, pero no bloqueará obstrucciones causadas externamente.

El i-gel no podrá insertarse si el paciente presenta **{% include hl.html t="Airway Inflammation" %} grave**.

# Uso
{% include git-wiki/components/template/action_note.html name="Insert i-gel" bodypart="Head" category="airway" %}

El i-gel puede retirarse en cualquier momento y se retirará automáticamente cuando el paciente despierte.
<br><br>
- **Recovery Position -** Si el paciente es colocado en posición de recuperación, insertar un i-gel cancelará la posición de recuperación.
- **{% include hl.html t="CPR" %} y {% include hl.html t="BVM" i="/wiki/image/bvm.png" %} -** Un i-gel insertado permite realizar simultáneamente RCP y ventilación con BVM.

## Indicaciones
- Paciente inconsciente sin dispositivo de vía aérea insertado
- Colapso grave de la vía aérea

## Contraindicaciones
- Inflamación grave de la vía aérea
