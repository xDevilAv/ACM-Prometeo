La **cánula nasofaríngea (NPA)** es un {% include hl.html t="Airway Adjunct" %} utilizado para mantener abierta la vía aérea superior en pacientes inconscientes.

# Función
El NPA se utiliza para asegurar el acceso a la vía aérea en caso de pérdida de reflejos protectores y {% include hl.html t="Airway Collapse" %} a través de la vía nasal.

Una vez insertado, impedirá establecer la {% include hl.html t="Recovery Position" %}.

El NPA tiene una capacidad limitada para mitigar el colapso de la vía aérea y deberá reemplazarse por una vía aérea avanzada si el paciente permanece inconsciente durante un periodo prolongado.

El NPA **no evita** las {% include hl.html t="Airway Obstructions" l="Airway_Obstruction" %}; si ocurren, deben ser despejadas.

# Uso
{% include git-wiki/components/template/action_note.html name="Insert NPA" bodypart="Head" category="airway" %}

El NPA puede retirarse en cualquier momento y se retirará automáticamente cuando el paciente despierte.

- **{% include hl.html t="Head Tilt-Chin Lift" %} -** Si el paciente presenta un colapso grave de la vía aérea, puede realizarse una maniobra de elevación del mentón con extensión de la cabeza con un NPA insertado para mitigar el colapso y mantener la vía aérea abierta.
- **Recovery Position -** Si el paciente se coloca en posición lateral de seguridad, insertar un NPA cancelará dicha posición.
- **{% include hl.html t="CPR" %} y {% include hl.html t="BVM" i="/wiki/image/bvm.png" %} -** Un NPA insertado **no permite** realizar simultáneamente RCP y ventilación con BVM.

## Indicaciones
- Paciente inconsciente sin otro dispositivo de vía aérea insertado

## Contraindicaciones
- Ya existe una vía aérea avanzada colocada
- Obstrucción de la vía aérea
- Colapso grave de la vía aérea
